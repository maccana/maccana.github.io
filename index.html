About

Relocated from Ireland to Quebec City, Canada, I am a web developer specialising in building web applications using modern open source technologies and frameworks.

I began experimenting with hacking early HTML in the late 1990's while completing a diploma in sound engineering. For several years I pursued a career in audio engineering, culminating in a MSc. in Music Technology, before returning to my passion for web programming. I was inspired by the huge advances in web and mobile technology and the explosion of the new powerful tools, made freely available to next generation developers.

I currently enjoy working with Javascript on both frontend (Angular.js, jQuery) and backend (Node.js, Express.js). I also have experience with a wide range of technologies covering all aspects of development such as Ruby on Rails, MySQL, MongoDB/NoSQL, Firebase, responsive design, Java and even have past experience with Android development.

Skills

I have a highly versatile skill set ranging from frontend HTML5, CSS3 and Javascript to backend technologies such as Node.js and Ruby on Rails. I love using open source technologies to craft modern applications with minimal and responsive UI.
